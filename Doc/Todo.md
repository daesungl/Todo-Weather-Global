- [ ] **선택적 소셜 연동(Identity Linking)**: 설정에서 구글/애플/이메일 계정을 연동하면 기존 익명 UID에 정보가 덧붙여져 데이터 이전 없이 영구 계정으로 전환되도록 지원.

- [ ] **FCM 푸시 알림 발송 아키텍처 구축**: 공유 플랜 업데이트 시 게스트 유저(익명 계정)에게도 푸시 알림이 갈 수 있도록 Supabase Edge Function 및 FCM 토큰 저장 로직 구현.
