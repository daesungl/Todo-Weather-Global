# Todo Weather Global - Getting Started

## 🌏 Project Overview
This is the global version of the **Todo Weather** application. It is designed to be premium, minimalist, and internationalization (i18n) ready.

## 🛠 Tech Stack
- **Framework**: Expo (React Native)
- **i18n**: `i18next`, `react-i18next`, `expo-localization`
- **Icons**: `lucide-react-native`
- **Animations**: `react-native-reanimated`
- **Design System**: Stitch-generated "Atmospheric Minimalist" (Blue/White theme)

## 📁 Project Structure
- `src/i18n`: Globalization settings. Use `en.json` and `ko.json` for translations.
- `src/theme`: Theme tokens based on the premium design system.
- `src/screens`: UI screens.
- `src/components`: Reusable UI components.

## 📚 Documentation
- [CODING_GUIDELINES.md](./CODING_GUIDELINES.md): 플랫폼별 개발 주의사항 및 코딩 규칙 가이드.

## 🚀 Running the Project
1. Install dependencies: `npm install`
2. Start the dev server: `npx expo start`

## 🎨 Design Vision
- **Atmospheric Editorial**: Inspired by the "Azure Horizon" design system, using bold typography and intentional asymmetry.
- **Glassmorphism**: Soft background blurs for primary weather cards.
- **No-Line Policy**: Use tonal shifts and white space instead of 1px borders for a premium feel.

## ✨ Core Features (Global Version)
- **Unified Weather & Todo**: Seamlessly manage your day with atmospheric weather context.
- **Travel Itinerary Weather**: Plan multi-destination trips and see forecasts for your entire route on a single timeline.
- **Multi-language Support**: Fully localized for Global users (EN/KO supported).
